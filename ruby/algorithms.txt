Release 0: How to use an elevator

Enter elevator and locate buttons for floors.
start from top, left corner. (This is floor 1 in the United States.)
  until the number of desired floor is encountered, 
 Count up by increments of 1 with each button.
  progress straight downwards until no more buttons can be found. 
    -If the correct number button was found, press the button when it is encountered. 
    - If the correct number button was not found, return to the top left button you started at without adding additional numbers to the count as you move. Move one button to the right, and increase count by one again.
      -after moving one button to the right, continue  progress straight downwards until no more buttons can be found.
        -repeat this process every time you reach the bottom of a column of buttons without encountering the correct number floor.
      -When the correct number button was found, press the button.


  
       






Release 1: How to find a specified name out of a list of names 

1.Identify the last name of a person. 
  2.Compare the first letter of the last name you wish to find with the first letter of the last name on the first entry. 
    3. When a match is found, continue to the next letter of the desired name with the next letter of the name on the list.
    - if that letter matches, continue to next letter. 
     -continue matching each letter in the last name until all letters have been matched 
        -if at any point the letters no longer match, progress to the next name on the list and start from step #2
    When all letters of the last name from the list have been matched to appear in the same order as the desired last name, continue process (starting from step 2) with the letters of the first name.
